Проект по регрессионному моделированию, посвящённый **прогнозированию цен на дома**

Проект фокусируется на **качественном анализе данных**, **сравнении различных семейств моделей**
и **обоснованном выборе финального решения**.

## Обзор проекта

Проект демонстрирует полный аналитический цикл работы с табличными данными:
от первичного анализа и проверки гипотез до сравнительной оценки моделей и формулирования выводов.

В рамках проекта выполнены:

- анализ распределения таргета и обоснование лог-трансформации цены;
- исследование числовых и категориальных признаков с использованием корреляций Пирсона и Спирмена, 
  а также меры связи eta-squared;
- сравнение нескольких семейств моделей с согласованными, модель-ориентированными пайплайнами предобработки, включая:
  - линейные модели (baseline и регуляризованные);
  - полиномиальную регрессию;
  - модели, основанные на расстояниях;
  - модели на основе деревьев решений;
- выбор финального решения с оптимальным балансом качества обобщения и интерпретируемости.

## Анализ данных и эксперименты

Вся работа над проектом — включая анализ данных, предобработку, моделирование и финальные выводы — 
выполнена в ноутбуке: `notebooks/house_price_predictor.ipynb`.

### Исследовательский и статистический анализ

В рамках данного раздела выполнены ключевые шаги анализа данных перед этапом моделирования:

- анализ структуры данных, типов признаков и характера пропусков в обучающей и тестовой выборках;
- анализ распределения целевой переменной и выбор лог-трансформации цены в соответствии с метрикой RMSLE;
- исследование распределений числовых признаков и их связей с целевой переменной;
- сравнение линейных и монотонных зависимостей для числовых признаков;
- анализ мультиколлинеарности между числовыми признаками и её влияния на выбор моделей;
- анализ категориальных признаков: распределение категорий и количественная оценка их связи с целевой переменной.

### Моделирование

В проекте проведено системное сравнение нескольких семейств регрессионных моделей с согласованными, 
модель-ориентированными пайплайнами предобработки и с общей метрикой оценки (RMSLE).

Рассмотрены следующие группы моделей:

- Линейные модели: OLS как baseline, а также Ridge, Lasso и Elastic Net для стабилизации линейной модели 
  в условиях высокой размерности и сильной коррелированности признаков.
- Полиномиальные модели: линейные модели с полиномиальным расширением признаков, дополненные PCA и регуляризацией.
- Модели, основанные на расстояниях: KNN и SVR для оценки применимости similarity-based подходов.
- Деревья и ансамбли: Decision Tree, Random Forest, XGBoost, LightGBM и CatBoost.

Для всех моделей выполнен подбор гиперпараметров с использованием кросс-валидации.
Оценка качества проводится в исходной шкале цен после обратного лог-преобразования.

### Ключевые выводы

- Регуляризованные линейные модели демонстрируют лучшее качество на отложенной выборке,
  что указывает на преобладание линейного сигнала в данных.
- Lasso-регрессия показывает минимальный RMSLE за счёт эффективного подавления шума 
  в высокоразмерном one-hot пространстве.
- Градиентный бустинг (CatBoost, LightGBM, XGBoost) демонстрирует сопоставимые результаты, 
  но не даёт прироста качества.
- Полиномиальные и distance-based модели теряют качество при росте размерности признаков.

## Запуск проекта

### Требования

- Python 3.9+ (протестировано на Python 3.12);
- Jupyter Notebook

### Установка и запуск

#### 1) Клонирование репозитория

```
git clone https://github.com/Lid11a/house-price-predictor
cd house-price-predictor
```

#### 2) Настройка токена Kaggle API (однократно)

Для автоматической загрузки датасета используется Kaggle API.

- **Windows:** `%USERPROFILE%\.kaggle\kaggle.json`
- **Linux / macOS:** `~/.kaggle/kaggle.json`

Для Linux / macOS также необходимо установить права доступа:

```
 chmod 600 ~/.kaggle/kaggle.json
```

#### 3) Создание и активация виртуального окружения и установка зависимостей

**Windows (PowerShell)**

```
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

**macOS / Linux**

```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

#### 4) Запуск анализа

Запустите Jupyter Notebook или Jupyter Lab и выполните ноутбук
`notebooks/house-price-predictor.ipynb` последовательно сверху вниз.

## Используемые технологии

В проекте применён набор инструментов, охватывающий полный цикл работы с табличными данными —
от статистического анализа и проверки гипотез до сравнительного моделирования и обоснованного выбора финального решения.

- **Анализ данных и статистика**  
  pandas, numpy — анализ распределений признаков, пропусков и целевой переменной, 
  проверка статистических зависимостей (в том числе eta-square тесты)

- **Визуализация и EDA**  
  matplotlib, seaborn

- **Линейные и регуляризованные модели**  
  Linear Regression (OLS), Ridge, Lasso, Elastic Net

- **Полиномиальные модели**
  Polynomial Regression в сочетании с PCA и регуляризацией

- **Методы, основанные на расстояниях и разделяющих поверхностях**  
  K-Nearest Neighbors, Support Vector Machines

- **Деревья решений и ансамблевые методы**  
  Decision Tree, Random Forest

- **Градиентный бустинг**  
  XGBoost, CatBoost, LightGBM

- **Инфраструктура экспериментов**  
  Jupyter Notebook
